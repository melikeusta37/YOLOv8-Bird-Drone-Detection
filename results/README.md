# Model Sonuçları

Bu klasör YOLOv8 modelinin eğitim sürecinde elde edilen performans çıktılarının saklanması amacıyla oluşturulmuştur.

İçerik:

- Confusion Matrix
- Normalize Edilmiş Confusion Matrix
- Precision Eğrisi
- Recall Eğrisi
- F1 Score Eğrisi
- Precision-Recall Eğrisi
- Eğitim Sonuç Grafikleri
