# YOLOv8-Bird-Drone-Detection
YOLOv8 based object detection system for bird and drone classification.

# YOLOv8 Kuş-Drone Tespiti

## Proje Hakkında

Bu proje, kuş ve insansız hava araçlarını (İHA/drone) ayırt edebilen YOLOv8 tabanlı bir nesne tespit sistemi geliştirmek amacıyla hazırlanmıştır.

Model, hazır bird-drone veri seti kullanılarak Google Colab ortamında eğitilmiş ve performansı çeşitli değerlendirme metrikleri ile analiz edilmiştir.

## Kullanılan Teknolojiler

* Python
* YOLOv8
* Ultralytics
* OpenCV
* Google Colab

## Özellikler

* Kuş ve drone tespiti
* Görüntü üzerinde nesne konumlandırma
* Güven skoru hesaplama
* Gerçek zamanlı tahmin desteği
* Performans analizi

## Sonuçlar

Eğitim sonucunda elde edilen performans grafikleri ve karışıklık matrisi (confusion matrix) proje içerisinde paylaşılmıştır.
